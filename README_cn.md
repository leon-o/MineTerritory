# Territory Mod of Minecraft
一个基于Forge(目前)的领地模组。

## 介绍

和其他领地模组和领地插件比起来，它有以下特点：

1. 保证游戏性，尽可能不破坏MC原版的Feel，领地台和附魔台是一对兄弟:smirk:
2. GUI界面，玩家可以通过GUI配置权限、领地区域、领地名字
3. 按照区块分领地，主要是为了性能考虑。获取玩家所在的区块坐标，到HashMap里一查就可以判断有没有权限。
